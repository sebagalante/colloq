# Custom Emojis

Place custom emoji PNG or SVG files in this directory.

## Required files for the default set:

- `racing.png` — Racing Club logo
- `boca.png` — Boca Juniors logo
- `river.png` — River Plate logo
- `independiente.png` — Independiente logo
- `sanlorenzo.png` — San Lorenzo logo
- `gol.png` — Goal celebration
- `roja.png` — Red card
- `amarilla.png` — Yellow card
- `var.png` — VAR check
- `penal.png` — Penalty
- `vamos.png` — Fan chant
- `daleacademia.png` — Racing chant
- `escudo.png` — Shield/crest
- `campeon.png` — Champion

## Recommended specs:
- Format: PNG with transparency or SVG
- Size: 64x64px or 128x128px
- Style: Flat, simple, recognizable at small sizes

## Usage in posts:
```
:racing: Dale :vamos:!
```
